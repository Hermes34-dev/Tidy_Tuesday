# TidyTuesday

My contributions to the [TidyTuesday]((https://github.com/rfordatascience/tidytuesday)) weekly data challenge,
built in Python.

TidyTuesday is an open data project by the Data Science Learning Community where
participants explore the same public dataset every week and share their visualisations.

## Tools
- Python 3
- pandas
- matplotlib
